📋 ПРОБЛЕМА:
───────────
Когда вы нажимаете "💾 Сохранить в GitHub" в админ-панели:
- Показывается: "✅ Успешно сохранено!"
- НО файл НЕ появляется в GitHub репозитории

💡 ПОЧЕМУ:
──────────
Не настроен GitHub токен в Cloudflare Pages.
Без токена API не может создавать файлы в репозитории.

═══════════════════════════════════════════════════════════════════════
    🚀 РЕШЕНИЕ (2 МИНУТЫ)
═══════════════════════════════════════════════════════════════════════

┌─ ШАГ 1: Создать GitHub токен ─────────────────────────────────────┐
│                                                                     │
│  1. Откройте: https://github.com/settings/tokens                   │
│                                                                     │
│  2. Нажмите: "Generate new token (classic)"                        │
│                                                                     │
│  3. Заполните:                                                     │
│     Note: youtube-feed-cloudflare                                  │
│     Expiration: No expiration                                      │
│     ✅ repo (поставьте галочку!)                                   │
│                                                                     │
│  4. Нажмите: "Generate token"                                      │
│                                                                     │
│  5. СКОПИРУЙТЕ токен (выглядит так):                               │
│     ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx                       │
│     ⚠️  Вы увидите его только один раз!                            │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

┌─ ШАГ 2: Добавить в Cloudflare ────────────────────────────────────┐
│                                                                     │
│  1. Откройте: https://dash.cloudflare.com/                         │
│                                                                     │
│  2. Перейдите:                                                     │
│     Workers & Pages → youtube-feed → Settings                      │
│                                                                     │
│  3. Найдите секцию: Environment variables                          │
│                                                                     │
│  4. Нажмите: "Add variables"                                       │
│                                                                     │
│  5. Добавьте ДВЕ переменные:                                       │
│                                                                     │
│     ┌───────────────────────────────────────────┐                 │
│     │ Variable name: GITHUB_TOKEN               │                 │
│     │ Value: [ваш токен ghp_xxx...]             │                 │
│     │ Environment: ✅ Production                │                 │
│     └───────────────────────────────────────────┘                 │
│                                                                     │
│     ┌───────────────────────────────────────────┐                 │
│     │ Variable name: GITHUB_REPO                │                 │
│     │ Value: TFWidgets/youtube-feed             │                 │
│     │ Environment: ✅ Production                │                 │
│     └───────────────────────────────────────────┘                 │
│                                                                     │
│  6. Нажмите: "Save"                                                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

┌─ ШАГ 3: Redeploy проекта ─────────────────────────────────────────┐
│                                                                     │
│  1. Перейдите на вкладку: Deployments                              │
│                                                                     │
│  2. Найдите последний деплой                                       │
│                                                                     │
│  3. Нажмите "···" (три точки) → "Retry deployment"                 │
│                                                                     │
│  4. Подождите ~2 минуты пока деплой завершится                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

┌─ ШАГ 4: Проверка ─────────────────────────────────────────────────┐
│                                                                     │
│  1. Откройте: https://youtube-feed.pages.dev/admin.html            │
│                                                                     │
│  2. Создайте тестовый конфиг:                                      │
│     Config ID: test123                                             │
│     Добавьте 1 видео URL                                           │
│                                                                     │
│  3. Нажмите: "💾 Сохранить в GitHub"                               │
│                                                                     │
│  4. Откройте GitHub:                                               │
│     https://github.com/TFWidgets/youtube-feed/tree/main/configs    │
│                                                                     │
│  5. ✅ Должен появиться файл: test123.json                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

❌ Показывает "✅ Успешно!" но файла нет в GitHub?
   → Проверьте, что добавили ОБЕ переменные
   → Проверьте, что сделали Retry deployment
   → Подождите 2 минуты после деплоя

❌ Ошибка 401?
   → Токен неправильный
   → Создайте новый токен с галочкой ✅ repo

❌ Ошибка 404?
   → GITHUB_REPO должен быть: TFWidgets/youtube-feed
   → НЕ просто: youtube-feed

❌ Конфиг сохранился, но не виден на сайте?
   → Cloudflare еще деплоится (подождите 1-2 минуты)
